# Use Cases

The following use cases show how FirecREST streamlines integration between cloud-ready applications and HPC resources. Choose one to work through during the hands-on session.

- [CI/CD Pipeline](ci-pipeline.md) — GitHub Actions workflow that runs tests on Piz Daint
- [Web UI](ui.md) — Flask application for submitting and monitoring HPC jobs
- [Airflow Operator](airflow.md) — Custom Apache Airflow operators backed by FirecREST
